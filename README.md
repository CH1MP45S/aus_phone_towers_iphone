## Getting Started

To use Cocopods on Windows: https://airtdave.medium.com/using-cocoapods-on-windows-dec471735f51

cd ios
gem install cocoapods -v 1.10.1
pod --version
1.10.1

You must use 1.10.1 because 1.11.* will be broken on Windows.

cd ios
flutter clean
flutter pub get
pod install
